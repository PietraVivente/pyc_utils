# pyc_utils

**Version:** 0.1

## Description
`pyc_utils` is a lightweight Python library that provides fast and convenient utilities, combining pure Python implementations with optional C extensions for performance.  
The library is designed to grow over time with new features, offering easy-to-use tools for common programming tasks.
